# 🍽️ Swiggy Restaurant Recommendation System

A Machine Learning-powered Restaurant Recommendation System built using Python, Scikit-Learn, K-Means Clustering, and Streamlit.

This project recommends similar restaurants based on cuisine and location by clustering restaurants into groups with similar characteristics.

---

## 📌 Project Overview

The goal of this project is to:

* Clean and preprocess restaurant data
* Transform categorical features using One-Hot Encoding
* Cluster restaurants using K-Means Clustering
* Build a recommendation engine
* Deploy recommendations through an interactive Streamlit application

---

## 🚀 Features

* Restaurant Recommendation Engine
* Data Preprocessing & Feature Engineering
* One-Hot Encoding
* K-Means Clustering
* Streamlit Web Application
* Model Persistence using Joblib

# 🔥 One-Hot Encoding

The categorical features:

* City
* Cuisine

were transformed into numerical representations using Scikit-Learn's OneHotEncoder.

# 📊 Encoded Feature Matrix

# 🤖 K-Means Clustering

Restaurants were grouped into clusters using MiniBatchKMeans.

This clustering approach helps identify restaurants with similar cuisine and location patterns.

![KMeans Clustering](images/kmeans_clustering.png)

# 🍴 Recommendation Engine

The recommendation function:

1. Finds the selected restaurant
2. Identifies its cluster
3. Retrieves restaurants from the same cluster
4. Returns the top recommendations

# 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Swiggy-Restaurant-Recommendation-System.git
```

Move into project directory:

```bash
cd Swiggy-Restaurant-Recommendation-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```
